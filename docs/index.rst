Pymacco
=======

Release v\ |version|

Pymacco is a version of the card game *Tomacco* (also known as 
`Shithead <http://en.wikipedia.org/wiki/Shithead_(card_game)>`_), written in Python.

.. warning::
    ``pymacco`` is currently a **WORK IN PROGRESS**. So much so, that it is not currently
    usable in any form.
